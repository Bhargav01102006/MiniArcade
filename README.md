# Mini Arcade 🕹️

A lightweight, offline gaming hub built in Python to bring classic entertainment straight to your PC—no internet or paper required!

## 🚀 Introduction
The **Mini Arcade** program is designed for quick entertainment whenever you are offline or taking a break. Instead of wasting paper or needing a second player, you can launch this program on your PC and start playing instantly.

### Available Games:
1. **Tic-Tac-Toe ❌⭕**: The classic game you know and love, built digitally so you can play without wasting paper.
2. **Stone, Paper, Scissors 🪨📄✂️**: No friend nearby? No problem. Test your luck and strategy against the computer.
3. **Dice Roller 🎲**: Forgot your physical dice for a board game? Don't worry, our built-in simulator has you covered.
4. **Hungry Snake 🐍**: *(Under Development)* — Coming soon in a future update!

---

## 🛠️ Tech Stack & Modules Used
This project leverages Python libraries and graphical frameworks to create user interfaces and interactive gameplay.

* **`tkinter`**: Used to design and build the primary graphical windows and navigation interfaces.
* **`pygame`**: Utilized to power the mechanics, rendering, and logic for Tic-Tac-Toe.
* **`turtle`**: Employed to handle the visual animations and graphics for the Dice roller.
* **`random`**: Powers the computer's randomized decision-making logic in Stone, Paper, Scissors.
